# Apple Silicon Path

> **TL;DR:** The Mac Studio M3 Ultra (256 GB unified memory, 819 GB/s) is the silent alternative. Runs MoE coding models at **69 tok/s** with zero build effort. Starting at [CHF 5,096](https://www.toppreise.ch/price-comparison/Complete-systems/APPLE-Mac-Studio-CTO-Apple-M3-Ultra-256GB-RAM-MU973-Z1CE-p829061) (256 GB, 1 TB) from Swiss retailers — ~CHF 800 below Apple MSRP. No M4 Ultra exists as of February 2026.

## Key specs — Mac Studio M3 Ultra

| Spec | Value |
|---|---|
| Memory | 256 GB unified (BTO) |
| Memory bandwidth | 819 GB/s |
| TDP | ~150W (near-silent) |
| MoE performance | 69 tok/s (GPT-OSS-120B), 84 tok/s (Qwen3-30B-A3B) |
| Dense 70B performance | ~15-20 tok/s (Q4) |
| Price (256 GB, 1 TB) | [CHF 5,096](https://www.toppreise.ch/price-comparison/Complete-systems/APPLE-Mac-Studio-CTO-Apple-M3-Ultra-256GB-RAM-MU973-Z1CE-p829061) (retail) / ~CHF 5,900 (Apple MSRP) |
